
## 关于

热爱前端，持续关注前端技术，包括但不限于：

- HTML / CSS / JavaScript

- Node.js
- 小程序开发
- 前端工程化
- AI应用开发


## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 预览构建结果
pnpm preview
```
